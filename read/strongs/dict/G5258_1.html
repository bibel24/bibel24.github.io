G5258

hupnos

Synonyme siehe: S0096

< ig.; vgl. "Hypnose" = schlafähnlicher Zustand; Subst.mask. (6)
I.) d. Schlaf
1) d. Schlummer; d. Tiefschlaf. Mt 1,24; Lk 9,32; Joh 11,13; Apg 20,9;
2) bildl. übertr.: von träger Untätigkeit. Röm 13,11;

Wortfamilie:

879 aph-upnoo
< 575 + 5258, (w. weg/ent-schlafen --> schläfrig machen/werden); Vb.(1)
I.) einschlafen
1) in Schlaf oder Schlummer fallen, schläfrig werden

69 agr-upneo
< Abl. von 71 (ergreifen, nachjagen) + 5258, (w. suchen-d. Schlaf); Vb.(4)
I.) schlaflos wachen
1) von jmd. d. eigentlich d. Schlaf sucht aber wachsam sein muß bzw.
wachsam bleiben will, wach bleiben, wachsam sein, aufpassen, sich
sorgen um jmd., über jmd. wachen, auf jmd. wachsam achthaben

70 agrupnia
< 69; Subst.fem. (2)
I.) Pl.: d. Schlaflosigkeiten
1) Schläfrigkeit, schlaflose Nächte (aufgrund von Sorge); d. Wachen,
d. Wachsamkeit.

1853 ex-upnos
< 1537 + 5258, (w. aus-d. Schlaf [aufgewacht); Adj. (1)
I.) aus d. Schlaf erwacht
1) aus d. Schlaf aufgeweckt --> wach sein/werden, aufwachen

1852 ex-upnizo
< 1537 + 5258, (w. aus-d. Schlaf [aufwachen]); Vb. (1)
I.) aus d. Schlaf erwecken
1) aus d. Schlaf holen --> aufwachen, wach werden.
