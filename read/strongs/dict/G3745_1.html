hosos

I.) so(...) wie...
1) von d. Ausdehnung d. Raumes oder d. Zeit, von d. Menge, d. Grad
oder d. Maß: so groß als; so weit als; so lang als; so viel als;
wie groß, weit oder lang; wie viel; um so mehr, soviel, in d. Maß.
Mt 9,15; Mk 2,19; Röm 7,1; Gal 4,1; Hebr 3,3; 10,37; Offb 18,7; 21,16;
uva.
II.) alle die...
1) von d. Anzahl oder d. Menge: alles was (auch immer)...;
Mt 14,36; 22,9; Mk 3,10; Apg 10,45; Joh 11,22; Röm 8,14; 2Tim 1,18;
uva.
2) mit () : alles, was auch immer...; alle welche.
Hi 1,12; Mt 13,46; 18,25; 28,20; Lk 4,40; Joh 10,8; Apg 3,24; 5,36; ua.
