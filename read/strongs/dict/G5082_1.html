G5082

telik-outos

< 3588 mit 2245 und 3778, korrelatives Demonstrativpron. zu 2245
Pron. (4)
Gräz. von Personen: in solchem Alter, so alt, so jung.
I.) so groß
1) von Dingen: von so gewaltiger Größe, so bedeutend.
2Kor 1,10; Hebr 2,3; Jas 3:4 Re 16:18
